# vue-apicloud
vue + apicloud

## 项目

* h5：vue + apicloud + aui
* android：android 原生
* ios：ios原生

## 运行

* 安装：
  * cd h5
  * yarn install
* 浏览器run：
  * yarn serve
  * open [http://localhost:8080/web_adapter/adapter.html](http://localhost:8080/web_adapter/adapter.html)
* android run：
  * yarn build:android
  * 在android studio中运行app
* ios run：
  * yarn build:ios
  * 在xcode中运行app
